# go-crawler-demo

Go crawler demo

# Clone

    mkdir -p ~/go/src/github.com/humbertodias && cd $_
    git clone https://github.com/humbertodias/go-crawler-demo
    cd go-crawler-demo

# Run

    make run

Check the files: oficinas.json, tramites.json
